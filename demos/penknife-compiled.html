This file loads the compiled Penknife code from a file called "fin/penknife-compiled.js" in the repo directory. To use this file, you must create penknife-compiled.js in one of two ways:

  • Install Node.js. From the repo directory, run the command "npm install" followed by the command "node build-era.js --build".
  • Modify penknife.html by uncommenting some lines that put the compiled string in a <textarea>. Then view that page, copy the string (which is currently over 500KB), and paste it into the proper file.
